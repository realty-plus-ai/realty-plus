# Realty+

AI-powered WhatsApp automation platform for real estate businesses.

Developed and operated by **Supreme Stud Private Limited**.

## Features

- Instant WhatsApp replies
- AI lead qualification
- Automated follow-ups
- CRM synchronization
- Property matching
- Buyer insights
- Broker notifications
- Analytics dashboard
- Responsive SaaS website
- SEO optimized
- GitHub Pages compatible

## Deployment

1. Upload all files to a GitHub repository.
2. Enable GitHub Pages.
3. Set the deployment branch to `main`.
4. Update:
   - `yourdomain.com`
   - email addresses
   - company details if required.
5. Publish.
